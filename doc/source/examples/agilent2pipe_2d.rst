convert example: agilent2pipe_2d
================================

This example shows how to use nmrglue to convert a 2D Agilent/Varian data set
into a NMRPipe file.

The data used in this example is available for
`download. <https://storage.googleapis.com/google-code-archive-downloads/v2/code.google.com/nmrglue/example_agilent2pipe_2d.zip>`_
