coadd example: coadd_2d_pipe
============================

This example shows how to use nmrglue to coadd a series of 2d NMRPipe files.
All files matching the ``test*.fid`` pattern will be coadded into a file
named ``coadded.fid``.

The data used in this example is available for
`download. <https://storage.googleapis.com/google-code-archive-downloads/v2/code.google.com/nmrglue/example_coadd_2d_pipe.zip>`_
